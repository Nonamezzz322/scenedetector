# SceneShot

Нативное macOS-приложение: извлекает кадры на **смене сцены** из видео (локальный файл или прямая ссылка на видеофайл) и сохраняет их картинками. Рассчитано на маркетологов — ставится как обычное приложение, консоль не нужна.

Движок — вшитый статический **ffmpeg** (фильтр `select=gt(scene,…)`), универсальный бинарь (Apple Silicon + Intel). Полный план реализации — в [PLAN.md](PLAN.md).

## Как пользоваться

1. Перетащите видеофайл в окно или нажмите «Выбрать видео…», **либо** вставьте прямую ссылку на видео.
2. (По желанию) раскройте «Расширенные настройки»: чувствительность, формат, папка, имена и т.д.
3. Нажмите «Извлечь кадры» — по готовности откроется папка с кадрами (по умолчанию `~/Movies/SceneShot/…`).

## Транскрипция голоса (вкладка «Транскрипция»)

Распознавание речи в текст **локально**, без интернета и аккаунтов — через встроенный **whisper.cpp**
(CPU) с моделью **ggml-base** (мультиязычная, есть русский).

1. Вкладка **«Транскрипция»** → перетащите видео/аудио или вставьте ссылку: прямой файл, **YouTube /
   YouTube Shorts / TikTok / Instagram Reels**, либо файл Dropbox/Google Drive.
2. Выберите язык (**Авто**/Русский/English) и форматы (**TXT** и/или **SRT**, оба по умолчанию).
3. Нажмите **«Транскрибировать»** → по готовности откроется папка с `transcript.txt`/`transcript.srt`,
   а в окне — превью текста с кнопкой «Копировать».

В пакетном режиме (вкладка «Папка») транскрипцию можно включить вместе с извлечением кадров —
для каждого видео создаётся подпапка с кадрами и/или транскриптом.

> Если whisper-модуль не собран, кнопка «Транскрибировать» неактивна, а в статусе видно
> «Модуль распознавания не установлен» — соберите его шагом `./Scripts/fetch-whisper.sh` ниже.

## Ссылки на соцсети (YouTube, TikTok, Instagram)

Во вкладках **«Кадры»** и **«Транскрипция»** можно вставить ссылку на страницу **YouTube / YouTube Shorts /
TikTok / Instagram Reels** — приложение само скачает видео (встроенный **yt-dlp**, использующий встроенный
ffmpeg) и обработает его как обычный файл. Ничего ставить пользователю не нужно.

> Ограничения честно: приватный/«только для друзей» контент и видео, требующие входа или подтверждения
> возраста, скачать нельзя. Эти площадки часто меняют внутренности — **yt-dlp нужно периодически
> обновлять** (`./Scripts/fetch-ytdlp.sh`, поднимите `VERSION` и пере-пинните sha256), иначе старая версия
> со временем перестаёт скачивать. Соблюдайте условия использования площадок и авторские права.

## Конкурент-аналитика (вкладка «Конкуренты»)

Сбор роликов конкурента и **метрики монтажа** — для насмотренности и анализа «залетающих» креативов:

1. Вставьте ссылку на **канал / профиль / хэштег / плейлист** (YouTube/TikTok/…) **или** поисковый запрос →
   нажмите «Найти» (через встроенный yt-dlp, лимит N настраивается).
2. Появляется сетка превью с галочками — отметьте нужные ролики.
3. По умолчанию считаются **метрики монтажа** (можно добавить кадры и/или транскрипт) → «Проанализировать».
4. Результат — таблица на каждое видео: **длительность, склейки, склейок/мин, средняя сцена (ASL), хук
   (время до 1-й склейки), слова** + средние по подборке. Кнопка **«Экспорт CSV»** кладёт `report.csv`
   (открывается в Excel/Sheets) в папку `~/Movies/SceneShot/<источник>-competitors-<дата>/`.

> Честно: метрики склеек — **аппроксимация ритма** (порог сцены ловит жёсткие переходы; быстрый
> монтаж/плавные переходы могут смещать счёт) — это индикатор темпа, не покадровый shot-detection.
> Массовый сбор зависит от площадки: Instagram-профили часто требуют входа, большие каналы — медленно
> (потому лимит N). yt-dlp нужно периодически обновлять. Соблюдайте ToS площадок и авторские права.

## Облако: Dropbox и Google Drive (вкладка «Папка»)

Кроме локальных файлов и прямых ссылок, приложение умеет работать со **ссылками Dropbox/Google Drive**:

- **Файл-ссылка** → вставьте во вкладку «Кадры» (одиночная обработка).
- **Ссылка на папку** → вкладка **«Папка»**: подгружается сетка превью всех видео, отмечаете нужные
  галочками («Выбрать все»), включаете «Извлечь кадры» и жмёте «Обработать выбранные» — результат
  складывается в подпапку на каждое видео. Локальные папки тоже поддерживаются (превью через QuickLook/ffmpeg).
- Подключение — в 1 клик («Подключить Dropbox»/«Google Drive»), вход через системный браузер
  (OAuth 2.0 + PKCE). Токены хранятся в файле `~/Library/Application Support/SceneShot/tokens.json`
  (права 0600, читает только ваш аккаунт) — **не в Keychain**, чтобы ad-hoc-сборка не спрашивала
  пароль связки ключей при каждом запуске. Кнопка «отключить» их стирает.

> **Транскрипция** в пакетном режиме зарезервирована (переключатель виден, но выключен) — появится
> после добавления модуля распознавания речи (см. `TRANSCRIPTION_PLAN.md`).

### Что нужно сделать разработчику один раз (для облака)

Облачные кнопки появляются только если в [Secrets.swift](Sources/SceneShot/Cloud/Secrets.swift) вписан **app key**
(публичный идентификатор — это НЕ секрет; `client secret` встраивать нельзя, у нас PKCE public client):

- **Dropbox** → [App Console](https://www.dropbox.com/developers/apps): scoped-приложение, включить PKCE,
  redirect URI `sceneshot://oauth`, scopes `files.metadata.read`, `files.content.read`, `sharing.read`;
  скопировать **App key** в `Secrets.dropboxAppKey`.
- **Google Drive** → Google Cloud Console: включить Drive API, OAuth client (Desktop), scope
  `drive.readonly`, redirect `sceneshot://oauth`, режим «Тестирование» + тест-пользователи; **Client ID**
  в `Secrets.googleClientID`.

Без ключей вкладка «Папка» работает только с локальными папками; облачные кнопки скрыты.

## Сборка (для разработчика)

Нужны только **Xcode Command Line Tools** (полный Xcode НЕ нужен):

```sh
./Scripts/fetch-ffmpeg.sh   # один раз: скачать ffmpeg/ffprobe (обе арх, проверка sha256)
./Scripts/fetch-whisper.sh  # один раз: собрать whisper-cli из исходников (нужен cmake) + модель ggml-base (~142 МБ)
./Scripts/fetch-ytdlp.sh    # один раз: скачать yt-dlp_macos (для ссылок YouTube/TikTok/Instagram), ~35 МБ
./Scripts/build.sh          # → dist/SceneShot.app (универсальный; FAST=1 — только хост-арх)
open dist/SceneShot.app
```

`fetch-whisper.sh` собирает универсальный CPU-only `whisper-cli` из исходников (Metal недоступен под CLT),
кладёт его в `Resources/Helpers/<arch>/` и скачивает модель в `Resources/Models/ggml-base.bin`
(пин по размеру+sha256). Нужен **cmake** (`brew install cmake`; Homebrew в `~/.brew`). Из-за модели
готовый `.app`/DMG вырастает примерно до **~280 МБ**. Проверить пайплайн: `./Scripts/test-transcribe.sh`.

Сборка идёт через `swiftc` + `lipo` с ручной сборкой `.app` (XcodeGen/`xcodebuild` не используются).

## Распространение

```sh
./Scripts/release.sh        # build → sign → dmg → notarize(-или-пропуск) → dist/SceneShot.dmg
```

**Без аккаунта Apple Developer** (текущее состояние): приложение подписывается ad-hoc. На **чужом**
Mac (особенно macOS 15 Sequoia / 26 Tahoe) Gatekeeper из-за карантина может показать **«не удаётся
проверить разработчика»** или даже **«повреждено, переместите в Корзину»** — файл при этом цел, это
защита, а не поломка. Снимается разово. Самый надёжный способ (работает и для «повреждено», где кнопки
«Открыть» нет):

```sh
xattr -dr com.apple.quarantine /Applications/SceneShot.app
```

Эта же инструкция (плюс вариант через Системные настройки) лежит в DMG — файл **КАК-ОТКРЫТЬ.txt**.
Сборка чистит расширенные атрибуты перед подписью и строго проверяет подпись (`codesign --verify
--deep --strict`), чтобы ad-hoc-печать не «ломалась» при переносе. Полностью без предупреждений у всех —
только с **нотаризацией** (нужен аккаунт Apple Developer; пайплайн уже готов, см. ниже).

**С аккаунтом Apple Developer** (чистый «поставил и работает», без предупреждений) — задайте
переменные окружения, код менять не нужно:

```sh
export CODESIGN_IDENTITY="Developer ID Application: Имя (TEAMID)"
export APPLE_ID="you@example.com"
export TEAM_ID="TEAMID"
export APP_PASSWORD="app-specific-password"
./Scripts/release.sh        # подпишет Developer ID + hardened runtime, нотаризует и застейплит DMG
```

## Иконка

`./Scripts/make-icon.sh` собирает `Resources/AppIcon.icns`. Положите свою картинку
`Resources/icon-source.png` (1024×1024) — иначе сгенерируется плейсхолдер-градиент.

## Лицензии встроенных бинарников

- **ffmpeg** — сборки с ffmpeg.martin-riedl.de (GPL). Приложение только декодирует видео и пишет
  JPG/PNG, но для публичной раздачи приложите LICENSE и предложение исходников, либо замените на
  LGPL-сборку (см. шапку `Scripts/fetch-ffmpeg.sh`). Текст — `Resources/FFMPEG-LICENSE.txt`.
- **whisper.cpp** — MIT (без copyleft), собирается из исходников. Модель ggml-base — OpenAI Whisper
  (MIT), конвертированная ggml-org. Текст и атрибуция — `Resources/WHISPER-LICENSE.txt`.
