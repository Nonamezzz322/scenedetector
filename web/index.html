Движок распознавания

Первый запуск скачивает модель распознавания (~80 МБ) с huggingface.co. Дальше работает офлайн из кэша браузера.

Прокси для ссылок (Dropbox / Google Drive)

Чтобы открывать видео по ссылкам с Dropbox/Google Drive (и прямым ссылкам без CORS), задеплойте бесплатный Cloudflare Worker (см. папку cloudflare-proxy) и вставьте его адрес сюда. YouTube/TikTok/Instagram так не работают.

О приложении

Всё происходит локально в браузере: ни видео, ни кадры, ни текст никуда не загружаются. Запись результата в выбранную папку доступна в Chrome/Edge; в других браузерах результат скачивается ZIP-архивом.
